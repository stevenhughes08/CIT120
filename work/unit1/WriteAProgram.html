The following are steps to writing a computer program.

  • Understand the program. What is the client asking for? What parameters and definitions does the program need to work in.
  • Plan the logic. This is an algorithm.
  • Code the program.
  • Use software (a compiler) to translate the progream into machine language.
  • Test the program, QA.
  • Put the program into production.
  • Maintain the program.
